# DATA

### Data Sources
- https://github.com/amir32002/3D_Street_View (nyc, chicago, sf, dc, vegas, florence, amsterdam, paris)
- https://www.crcv.ucf.edu/data/GMCP_Geolocalization/ (pittsburg, orlando, nyc)
- https://svsdataset.github.io/ (London, Paris, Madrid, Boston, San Francisco, Mexico City, Rio de Janeiro, Sydney, Bangkok, Tokyo)
- https://www.kaggle.com/datasets/paulchambaz/google-street-view (missing Canada, USA, Norway, France, Germany, Vietnam, and New Zealand)
- https://huggingface.co/datasets/blalexa/google-streetview-panoramas-geotagged
- https://huggingface.co/datasets/everettshen/StreetView360AtoZ (random)
- https://huggingface.co/datasets/stochastic/random_streetview_images_pano_v0.0.2 (random)
